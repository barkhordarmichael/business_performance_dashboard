# Business Performance Dashboard

A modern, responsive web-based dashboard for monitoring and visualizing business performance metrics.

## Overview

This dashboard provides a comprehensive view of key business metrics including revenue, growth rates, customer acquisition, and operational efficiency. Built with modern web technologies for a clean, intuitive user experience.

## Features

- **Interactive Charts**: Visualize data with various chart types using Chart.js
- **Performance Gauges**: Circular gauges for quick metric assessment
- **Responsive Design**: Optimized for desktop and mobile devices
- **Data Export**: Export dashboard data to Excel format
- **Data Import**: Import data from JSON files
- **Real-time Updates**: Dynamic data visualization with smooth animations
- **Custom Styling**: Tailwind CSS with custom color scheme

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with Tailwind CSS framework
- **JavaScript**: Interactive functionality
- **Chart.js**: Data visualization library
- **SheetJS (XLSX)**: Excel file handling
- **Tailwind CSS**: Utility-first CSS framework
- **Inter Font**: Modern typography

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/barkhordarmichael/business_performance_dashboard.git
   ```

2. Navigate to the project directory:
   ```bash
   cd business_performance_dashboard
   ```

3. Open `index.html` in your web browser

### Usage

- **View Metrics**: The dashboard displays key performance indicators
- **Export Data**: Click the "Export" button to download data as Excel file
- **Import Data**: Click the "Import" button to load data from a JSON file
- **Responsive**: Resize your browser or view on mobile devices
